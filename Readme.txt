1. Open the fouanguage.exe 

2. Select 'Encrypt' or 'Decrypt' according your need.

3. Now enter the message in message section.

4. Enter the encryption key if you want to decrypt your message.
   You can use the same key to encrypt multiple  message by entering the key while encrypting message.
   If you want new key then leave the key section blank while encrypting.

5. Click 'Generate' to see the result.

6. To save the encrypted or decrypted file as text file click 'Save file'. A file with the name 'Encryted 
   message.txt' or 'decrypted message.txt' will be saved in the same directory as the app.
